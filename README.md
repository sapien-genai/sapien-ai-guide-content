# Sapien AI Integration Guide Web App

An interactive web application for the AI Integration Guide by Tony Marks.

## Features
- 30-day AI integration challenge
- Interactive exercises and quizzes
- Progress tracking
- Prompt library
- User profiles
- Note-taking capabilities

## Tech Stack
- React
- Tailwind CSS
- Shadcn/ui
- Supabase (Database)
- Stripe (Payments)
- Claude API (Chatbot)
